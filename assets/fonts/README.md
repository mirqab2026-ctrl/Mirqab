# خطوط مرقاب · Mirqab Fonts

## الخط السعودي · Saudi Font

**الخط السعودي** هو الخط الرسمي للمملكة العربية السعودية.
الملفات المُحمّلة حالياً:

| الوزن | اسم الملف | الحجم |
|--------|------------|--------|
| Regular (400) | `Saudi-Regular.ttf` | 104 KB |
| Bold (700) | `Saudi-Bold.ttf` | 125 KB |

### كيف يعمل:

1. النظام يقرأ ملفات الخط من هذا المجلد.
2. يُضمّنها كـ **base64** داخل CSS عبر `@font-face`.
3. يعمل **بدون أي اتصال إنترنت** (air-gapped).
4. اسم الـ font-family في CSS هو `'Saudi'`.

### الأسماء المتوقعة (للتمديد المستقبلي):

النظام يبحث عن أحد هذه الملفات تلقائياً:

```
Saudi-Light.ttf       (300)
Saudi-Regular.ttf     (400)  ← موجود ✅
Saudi-Medium.ttf      (500)
Saudi-Bold.ttf        (700)  ← موجود ✅
Saudi.ttf             (400 - fallback)
```

كما يدعم النظام صيغة `.otf` بنفس الأسماء.
